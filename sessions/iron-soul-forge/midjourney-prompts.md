## Usage Notes

1. Generate images in sequence to match the journey narrative
2. Use `--ar 16:9` for video backgrounds (1920x1080)
3. Use `--v 6.1` for consistent style
4. Style parameter `--s 750` balances creativity with prompt adherence
5. Consider generating 2-3 variations of each and selecting the best
6. For thumbnail, may want to add `--style raw` for more photorealistic rendering
