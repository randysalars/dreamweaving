## Additional Notes for Image Generation

### Style Consistency
- Maintain warm golden/amber color palette throughout
- Use consistent stone texture for the steps
- Keep mist/cloud elements present in most images
- Eastern temple architecture should be consistent
- Morning/sunrise lighting in all images

### Required Aspect Ratio
All images should be generated at **16:9** aspect ratio for video compatibility.

### Post-Processing Suggestions
- Consider slight vignette to focus attention
- Ensure faces are not too detailed (universal identification)
- Match color grading across all images

### Upload Instructions
1. Generate all images using the prompts above
2. Upscale to at least 1920x1080 resolution
3. Save as PNG files with the specified filenames
4. Place in `sessions/temple-of-a-thousand-steps/images/uploaded/`
