# Cookbook Template

Create a practical recipe/meal guide that provides genuine value for the reader's nutrition journey.

## Required Sections

### Introduction
- Explain the cooking philosophy (simple, whole foods, minimal time)
- State how recipes align with the main product principles
- Overview of what's included

### Weekly Shopping Lists (4 weeks minimum)
For each week:
- Proteins with quantities
- Vegetables with quantities  
- Grains/Carbs
- Pantry staples needed
- Estimated weekly cost ($XX-$XX range)

### Complete Recipes (12-16 recipes minimum)
For each recipe include:
- **Recipe Name**
- **Prep Time / Cook Time**
- **Servings**
- **Protein per serving (grams)**
- **Ingredients** (with specific measurements)
- **Instructions** (numbered steps)
- **Make-Ahead/Storage Tips**

Recipe categories:
- Breakfasts (3-4 recipes)
- Lunches (4-5 recipes)
- Dinners (4-5 recipes)
- Quick options (2-3 recipes)

### Meal Prep Guide
- Sunday prep routine with timing
- Storage guidelines
- Batch cooking strategies

### Quick Reference
- Protein sources table with amounts
- Simple vegetable sides (one-paragraph each)

## Quality Standards

- Recipes must be genuinely achievable (30 min or less)
- Real, specific ingredients (not "vegetables" but "2 cups broccoli florets")
- Actual cooking instructions, not summaries
- Target: 2000+ words minimum
