# Guided Meditation/Script Template

Create complete word-for-word scripts that the reader can follow or record.

## Required Sections

### Introduction
- How to use the scripts
- Recording tips (for self-recording)
- Reading pace and pause guidance
- Environment preparation

### Complete Scripts (5 minimum)
Each script must include:

**Script Format:**
```
# [Meditation Name]

Duration: [X-Y] minutes

## The Script

[Complete word-for-word script]

Use ... for pause indicators (2-3 seconds)
Use [longer pause] for extended pauses

Include:
- Opening/settling instructions
- Main practice content
- Closing/return instructions
```

Suggested scripts:
- Foundation/Beginner practice (10-15 min)
- Body awareness or body scan (15-20 min)
- Breath-focused (10-15 min)
- Emotional/loving-kindness (12-15 min)
- Sleep preparation (10-15 min, designed for bed)
- Short/emergency (5-8 min, acute stress)
- Morning/intention-setting (5-8 min)

### Script Components

Each script should contain:
- Settling phase (1-2 min)
- Main practice (varies by type)
- Integration/closing (1-2 min)
- Return to present instructions

## Format Requirements

- Write in second person ("You are...")
- Use ellipses (...) for natural pauses
- Include [bracketed instructions] for timing
- Write at reading pace (about 120 words per minute)
- Separate script sections clearly

## Quality Standards

- Scripts must be complete (not outlines)
- Pacing should feel natural when read aloud
- Language should be calming but not overly flowery
- Each script serves a distinct purpose
- Target: 2000+ words minimum
