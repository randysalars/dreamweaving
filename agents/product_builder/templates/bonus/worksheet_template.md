# Worksheet Template

Create structured self-assessment and planning worksheets that guide the reader through practical exercises.

## Required Sections

### Introduction
- How to use the worksheets
- Recommended timing (when to complete each)
- Tips for honest self-assessment

### Baseline Assessment Worksheet
- Rating scales for each pillar/area (1-10)
- Current habits inventory (Yes/No/Sometimes checkboxes)
- Goals identification (3-5 primary goals)
- Obstacles anticipation
- Motivation statement (fill-in)

### Weekly Planning Worksheet
- Previous week review prompts
- Daily schedule templates
- Priority identification
- Obstacle planning

### Progress Assessment Worksheet (30/60/90 day versions)
- Before/after comparison scales
- Habit consistency tracking
- Pillar assessments
- Wins documentation
- Next phase focus

### Specialized Audit Worksheets (2-3 minimum)
For problem areas, create:
- Current patterns assessment
- Environment audit
- Disruptor identification checklist
- Action plan template

### Completion/Final Worksheet
- Overall transformation summary
- Practices now automatic
- Areas needing attention
- Maintenance plan
- Final reflection prompts

## Format Requirements

Each worksheet should have:
- Clear section headers
- Rating scales (1-10 with labels)
- Fill-in-the-blank lines (represented as _____)
- Checklists where appropriate
- Space indicators for written responses

## Quality Standards

- Focus on self-discovery, not just tracking
- Questions should prompt genuine insight
- Each worksheet serves a distinct purpose
- Target: 1500+ words minimum
