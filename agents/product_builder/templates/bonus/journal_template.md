# Journal Template

Create a structured journaling system with evidence-based prompts for daily reflection.

## Required Sections

### Introduction
- Research supporting journaling benefits
- How to use this journal
- Daily time investment
- Daily prompt structure (e.g., Reflection/Gratitude/Intention)

### Daily Prompts by Phase

Structure prompts around the main product phases:

**Phase 1 (Days 1-30): Foundation**
- Days 1-7: Complete daily prompts with themes
  - Each day has 3 prompts
  - Day 7 is weekly review
- Days 8-14: Focus area prompts
- Days 15-21: Focus area prompts  
- Days 22-30: Integration prompts
- Day 30: Monthly assessment template

**Phase 2 (Days 31-60): Expansion**
- Overview of phase focus
- Weekly themes
- Key prompt categories

**Phase 3 (Days 61-90): Integration**
- Overview of phase focus
- Weekly themes
- Final consolidation prompts

### Weekly Review Template
Standard questions for every week:
- Wins
- Challenges
- Patterns noticed
- Insights
- Adjustments
- Ratings (numbered scales)

### Monthly Assessment Template
Deeper reflection questions for end of each month

### Special Situation Prompts
Prompts for:
- When falling off track
- When discouraged
- When overwhelmed
- Celebrating milestones

### Gratitude Prompt Library
Categories of gratitude prompts:
- Body gratitude
- Relationship gratitude
- Simple pleasures
- Growth gratitude

## Format Requirements

- Each prompt should be specific, not generic
- Include the "why" briefly where helpful
- Use fill-in sections for active journaling
- Prompts build on each other across days

## Quality Standards

- Prompts based on research (cite sources in intro)
- Variety prevents boredom
- Prompts should lead to genuine insight
- 30+ unique daily prompts minimum
- Target: 1500+ words minimum
