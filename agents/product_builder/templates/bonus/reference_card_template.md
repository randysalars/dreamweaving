# Reference Card Template

Create quick-reference cards that distill protocols and key information into scannable, actionable formats.

## Required Sections

### Introduction
- How to use the cards
- Printing/saving recommendations
- When to reference each card

### Protocol Cards (4-6 minimum)
For each major protocol/routine, create a card with:

**Card Format:**
```
# [Protocol Name] Card

## [Time Period or Trigger]

**Section 1:**
- Bullet points with specific actions
- Times or durations where relevant
- Key metrics to hit

**Section 2:**
[Continue with structured content]

## [Minimum Viable Version]
When time is short, do at least this...
```

Suggested cards:
- Morning Protocol
- Evening Protocol  
- Nutrition Essentials
- Movement Hierarchy
- Mindfulness Toolkit
- Habit Formation
- Phase Transitions
- Emergency/Crisis Tools

### Technique Quick Reference
For any key techniques, include:
- Name and alternate names
- Brief description (1 sentence)
- Step-by-step instructions
- When to use
- Duration

### Tables and Checklists
Include formatted tables for:
- Key metrics and targets
- Quick comparison data
- Checklist items

## Format Requirements

- Use bold headers for scanning
- Short bullet points (not paragraphs)
- Specific numbers and metrics
- Clear section breaks between cards

## Quality Standards

- Information must be instantly actionable
- No explanation needed (that's in main product)
- Print-friendly formatting
- Target: 1200+ words minimum
