### Core Structure Requirements

- Standard session length: 20-45 minutes total
- Include induction (5-10 minutes) - already writen as a template and will be added later
- Main content/journey (10-30 minutes) - to be written now
- Re-emergence/closing (5-7 minutes)- already writen as a template and will be added later
### Key Components

- Multi-sensory descriptions (visual, auditory, kinesthetic, etc.)
- Embedded commands and suggestions
- Natural pacing with appropriate pauses
- Safety elements and ethical considerations
## AI Script Generation Instructions Template

When asking an AI to generate a hypnosis script, include these specifications:

1. "Please create a hypnosis script that:"
1. Has a specific purpose/goal (specify the intended outcome)
1. Follows this structure:

  - Opening/Induction phase
  - Main content/journey phase
  - Re-emergence/closing phase
1. Includes:

  - Multi-sensory descriptions
  - Appropriate pacing and pauses
  - Embedded suggestions
  - Safety elements
1. Uses hypnotic language patterns:

  - Permissive language
  - Future pacing
  - Embedded commands
  - Presuppositions
1. Maintains ethical guidelines:

  - No harmful suggestions
  - Respect for autonomy
  - Clear boundaries
  - Professional limitations
The AI should also be instructed to mark pauses, tone changes, and emphasis points in the script as per elevenlabs commands.
