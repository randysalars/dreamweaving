# YouTube Thumbnail Design Guide

Expert-level thumbnail design principles for Sacred Digital Dreamweaver videos.

---

## Core Principle: Instant Clarity at 200x112 Pixels

Most viewers see thumbnails at extremely small sizes. Design for **recognition within 0.2 seconds**.

- Keep **one dominant subject**
- Use **bold, simple shapes** with obvious edges
- Avoid clutter - every extra detail reduces CTR

---

## The 10 Commandments of High-CTR Thumbnails

### 1. Strong Visual Hierarchy

Use **Gestalt** and **focal point theory**:

| Priority | Element | Purpose |
|----------|---------|---------|
| 1st | Large face or central object | Immediate attention grabber |
| 2nd | Bold title text (2-5 words max) | Communicate value |
| 3rd | Environmental/symbolic elements | Set mood and context |

Eye-tracking research shows viewers follow: **Face → Text → Symbol → Background**

### 2. Faces Always Win (With BIG Emotions)

For sessions with human elements, use:
- Exaggerated emotional expression (wonder, peace, transcendence)
- Large, clear eyes
- Clean edge separation from background

For abstract spiritual content:
- Central luminous focal point (portal, light source, sacred symbol)
- "Face-like" compositional weight in the image center

### 3. Curiosity Gap + Implied Story

A thumbnail should **raise a question but not answer it**.

Best formats for Dreamweaver:
- "What lies beyond?" portal imagery
- Mysterious light emanating from unknown source
- Figure approaching something transformative
- Symbolic object with unexplained power

### 4. High Contrast Color Blocking

| Color Strategy | When to Use |
|----------------|-------------|
| Hot accents (gold, amber, warm white) | Spiritual awakening, divine light |
| Cold backgrounds (deep blues, cosmic black) | Makes focal elements pop |
| Complementary pairs (gold/blue, purple/amber) | Maximum visual impact |

Avoid low-contrast palettes unless intentional for specific mood.

### 5. Short Text, Big Font, Zero Fluff

Text is **supporting**, not **explaining**:

| Guideline | Specification |
|-----------|---------------|
| Word count | 2-5 punchy words maximum |
| Font style | Sans-serif, bold, thick strokes |
| Contrast | White on dark, gold on dark |
| Effects | Subtle shadow/glow for edge visibility |

**Never use thin fonts or fine details in text.**

### 6. Mobile-First Design

90% of first impressions come from tiny previews.

Mobile rules:
- Subject fills **40-60%** of the frame
- No thin fonts
- No fine details that disappear at small sizes
- No wide-angle compositions

### 7. Pattern Disruption

Thumbnails that look visually different from surrounding videos get more clicks.

Options for Dreamweaver:
- Ethereal color palettes that stand out
- Luminous center with dark edges
- Unexpected sacred geometry
- High-contrast silhouettes with inner glow

### 8. Pose Language (For Human Subjects)

If using human figures:
- Serene, eyes-closed meditation pose
- Arms open in receptive gesture
- Profile gazing toward light
- Hands in mudra position

### 9. Consistent Branding

Maintain across all thumbnails:

| Element | Standard |
|---------|----------|
| Color palette | Sacred Digital Dreamweaver signature colors |
| Font family | Consistent across all sessions |
| Glow/lighting style | Ethereal, luminous, mystical |
| Logo placement | Subtle, non-intrusive |

### 10. A/B Testing Mindset

Test variations for:
- Color scheme shifts
- Text phrasing (e.g., "Journey" vs "Meditation" vs "Experience")
- Curiosity gap strength
- Background complexity vs solid color

## Text Styling Guidelines

### Title Text

```python
# Primary title properties
font_family = "DejaVuSans-Bold" or "Montserrat-Bold"
font_size = 80-120px  # Scales with thumbnail
color = "#FFFFFF"  # Pure white
shadow_color = "#000000"  # Black with 60-80% opacity
shadow_offset = (4, 4)  # Pixels
glow_color = session_accent_color  # Theme-appropriate
glow_radius = 8-12px
```

### Subtitle/Tagline

```python
# Secondary text properties
font_family = "DejaVuSans" or "Montserrat-Regular"
font_size = 32-48px
color = "#E8E8E8"  # Slightly off-white
shadow_color = "#000000"
shadow_offset = (2, 2)
```

### Badges (Duration, Features)

```python
# Badge properties
background_color = rgba(0, 0, 0, 0.8)  # Semi-transparent black
border_radius = 8px
padding = (15, 10)
font_size = 28-35px
text_color = "#FFFFFF"
border = "2px solid accent_color"  # Optional accent border
```

## Technical Specifications

### Output Requirements

| Property | Specification |
|----------|---------------|
| Dimensions | 1280 x 720 pixels |
| Aspect Ratio | 16:9 |
| Format | PNG (for quality) or JPEG (quality=95) |
| Color Space | sRGB |
| File Size | Under 2MB |

│   │                                                       │   │
│   │   FOCAL ZONE (center 40%)                            │   │

## Integration with Session Workflow

### Thumbnail Generation Trigger

Thumbnails should be generated during the `/build-video` or `/full-build` workflow.

### Input Requirements

1. **Session manifest** (`manifest.yaml`) - for title, theme, duration
2. **Base image** - either:
   - User-uploaded image from Midjourney (`images/uploaded/`)
   - Auto-generated from session imagery
3. **Session metadata** - for badges (Hz, duration, features)

### Output Location

```
sessions/{session-name}/
├── output/
│   ├── youtube_thumbnail.png      # Main thumbnail
│   └── youtube_package/
│       └── thumbnail.png          # Copy for package
```

## Quality Checklist

Before finalizing a thumbnail:

- [ ] Readable at 200x112 pixels (mobile preview size)
- [ ] Title visible and legible
- [ ] Single clear focal point
- [ ] High contrast between elements
- [ ] Consistent with channel branding
- [ ] No critical content in outer 10% margins
- [ ] Duration badge doesn't overlap with YouTube's timestamp
- [ ] Colors pop against YouTube's dark/light modes
- [ ] File under 2MB
- [ ] Evokes curiosity without misleading
